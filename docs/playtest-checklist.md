# Playtest Checklist

## Open

- PC: `http://localhost:3001`
- Smartphone: `http://192.168.1.6:3001`

## Check

- [ ] 最初は部屋作成/join画面だけが出る
- [ ] スマホ縦持ちでもルーム作成画面が横向きに見える
- [ ] PCで `CREATE ROOM` を押すとroom codeが出る
- [ ] room code入力で文字が二重表示されない
- [ ] room code入力中に入力不能にならない
- [ ] スマホで同じroom codeを入力して `JOIN` できる
- [ ] 2人揃うとゲーム画面の `READY` に移る
- [ ] 各画面に `CHUSER` / `DODGER` のどちらかが表示される
- [ ] 2人の役割が同じにならない
- [ ] タップしても各自で勝手にゲーム開始しない
- [ ] `START READY` を押すと `READY 1/2` になる
- [ ] 2人とも `START READY` を押すとゲーム開始する
- [ ] 開始後に `TIME` が表示されて動ける
- [ ] 右上に `5 / 4 / 3 / 2 / 1` のカウントが出る
- [ ] DODGER役には `ESCAPE!!` が小さく表示される
- [ ] CHUSER役には `HOLD YOUR CHU...` が小さく表示される
- [ ] CHUSERが透明カプセルに閉じ込められている
- [ ] 5秒後に表示が `CHASE!` へ切り替わる
- [ ] 開始後5秒間は逃げ役だけ動ける
- [ ] PCで動くとスマホ側にも位置が反映される
- [ ] スマホで動くとPC側にも位置が反映される
- [ ] 相手キャラが画面内で見える
- [ ] 移動中、自分のキャラが画面中心付近にいる
- [ ] 逃げられる範囲が広くなっている
- [ ] 鬼が逃げ役に触れると試合が終了する
- [ ] 捕獲時にカメラが寄る/揺れる
- [ ] 捕獲時に `BUCHU!!` と `!?` が目立つ
- [ ] 終了後に `YOU WIN!` / `YOU LOSE!` が表示される
- [ ] スマホ縦持ちの終了後テキストは横向き表示になる
- [ ] 終了後に `REMATCH READY` が表示される
- [ ] スマホ縦持ちの `REMATCH READY` まわりも横向き表示になる
- [ ] 2人とも `REMATCH READY` を押すと再スタートする
- [ ] スマホ縦持ちで横画面として自然に見える
- [ ] PC横幅768px以下でプレイ不可表示になる
- [ ] 見た目が大きく崩れていない
